# 🎛️ 预设助手 (ST-Presets-Assistant)

> SillyTavern 预设条目的 AI 可视化控制面板

---

## 这是什么？

**预设助手**是一个 SillyTavern 酒馆助手脚本，它能把你预设里密密麻麻的条目（那些你每次都要翻半天才找到的开关）变成一个**漂亮的毛玻璃控制面板**。

你只需要点一下，AI 就会自动分析你的预设结构，生成一个分类清晰、一目了然的控制台——不用再在预设编辑器里上下翻找了。

### ✨ 核心卖点

- **AI 智能分组** — 自动识别作者的分区标记、emoji 前缀、互斥组，生成合理的卡片布局
- **一键开关** — 直接在面板上切换预设条目，实时生效
- **条目绑定** — 把必须同时开关的条目绑在一起（比如「病名为爱」和「纯爱思维链」）
- **每预设独立保存** — 切换预设时自动恢复之前的面板配置
- **可视化编辑** — 不满意 AI 的布局？进入编辑模式自己拖拽调整
- **导入导出** — 和朋友分享你精心调整的面板配置

---

## 安装

### 前置要求

- SillyTavern（需要安装酒馆助手/TavernHelper）
- 一个可用的 AI API（用于生成面板布局）

### 安装步骤

1. 打开 SillyTavern 的**酒馆助手**设置
2. 导入 `酒馆助手脚本-预设控制.json`
3. 确保脚本已启用
4. 刷新页面，看到右下角的 🎛️ 悬浮球就说明安装成功了

---

## 使用指南

### 🟢 基本操作

#### 打开面板

- 点击右下角的**悬浮球**
- 选择 ⚙️ 齿轮按钮打开完整控制面板
- 或者在酒馆助手的魔法棒菜单里点「预设控制」

#### 生成控制台

1. 先确保你已经加载了一个预设（比如明月秋青、Kemini 等）
2. 打开面板，点击底部工具栏的 **🔄 重置基础模式** 按钮
3. AI 会分析你的预设条目，自动生成一个分组好的控制面板
4. 等待几秒，毛玻璃卡片布局就出来了！

#### 切换条目

- 直接点击 toggle 开关即可切换预设条目的启用/禁用状态
- 修改**实时生效**，不需要额外保存

---

### 🔗 条目绑定

有些条目需要同时开关（比如「病名为爱」必须搭配「纯爱思维链」）。

**AI 自动识别**：如果作者在名称里标注了「搭配xx一起开」「必须开xx」「需要搭配xx」，AI 会自动设置绑定。

**手动绑定**：
1. 进入**编辑模式**（点击工具栏的 ✏️ 笔按钮）
2. 找到要绑定的 toggle，点击旁边的 **🔗 链接按钮**
3. 在弹出的面板中：
   - 用**搜索框**快速找到目标条目
   - **勾选**要绑定的条目
   - 已绑定的条目会**高亮置顶**
4. 绑定完成后，切换一个条目时所有绑定条目会同步开关
5. 需要取消？点「**全部解绑**」或逐个取消勾选

---

### ✏️ 编辑模式

点击工具栏的 ✏️ 进入编辑模式，可以：

| 操作 | 说明 |
|------|------|
| ⬆️⬇️ 上下移动 | 调整条目在卡片内的顺序 |
| 🔗 绑定 | 管理条目间的联动关系 |
| 🗑️ 删除 | 移除不需要的条目 |
| 编辑文字 | 双击 card 标题可以改名 |
| 调整大小 | 拖拽 card 边缘可以调整卡片尺寸 |

### 💬 AI 对话

面板左侧是一个**聊天区域**，你可以和 AI 对话来调整面板：

- 「把文风相关的条目放在一起」
- 「给 NSFW 的卡片换个颜色」
- 「帮我重新整理一下布局」

AI 会根据你的指令修改面板配置。

---

### 📦 导入 / 导出

在工具栏中：

- **📤 导出** — 将当前面板配置保存为 JSON 文件（不含 API 密钥等敏感信息）
- **📥 导入** — 从 JSON 文件导入别人分享的面板配置

> 导入前会自动保存当前配置为快照，可以随时回滚。

---

### 📸 历史快照

每次 AI 生成或导入配置时，都会自动保存一份快照。点击工具栏的 **🕐 历史** 按钮可以：

- 查看所有保存的快照
- 预览每个快照的面板外观
- 一键回滚到任意历史版本

---

### ⚙️ API 配置

面板支持两种 API 模式：

- **酒馆模式**（默认）— 使用 SillyTavern 自带的 API 设置
- **自定义模式** — 手动配置 API 地址、密钥和模型

在面板设置区域可以切换和配置。

---

### 🎨 主题

支持两套主题：

- **🌙 暗色模式**（默认）— 深色背景 + 毛玻璃效果
- **📜 羊皮纸模式** — 温暖的古典风格

## 常见问题

**Q: AI 生成的布局不好看怎么办？**
> 进入编辑模式手动调整，或者在聊天里告诉 AI 怎么改。

**Q: 切换预设后面板配置丢了？**
> 不会。每个预设的配置独立保存，切回来会自动恢复。

**Q: 可以和朋友分享面板吗？**
> 可以！用导出功能生成 JSON 文件发给朋友，对方导入即可。导出不包含 API 密钥。

**Q: 绑定的条目取消绑定后还会联动吗？**
> 不会。解绑后各条目独立操作。
